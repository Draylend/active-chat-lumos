# active-chat
Team: [Draylend Chow](https://github.com/Draylend), [Sky Huang](https://github.com/kshuang04), [Keena Lin](https://github.com/keenalin), and [Lahari Karmuchi](https://github.com/Lkarm003)

A front-end framework and web back-end to support the embedding of activities into chat-like interfaces.

CS179M Senior Project for AI-Assisted Active Testing.

Our team is working with [Joshua Zingale](https://github.com/joshua-zingale) to develop the front-end and back-end for the **active-chat** feature.

Tech stack: JavaScript (for front-end) and Go (for back-end server)

See the [development plan](docs/development/DEVELOPMENT_PLAN.md).

# How to Run our Project
Run the following commands in your terminal
- in the `component-server/` directory, run `go run cmd/registry/main.go` to start up the component server
- run `npx serve` and navigate to the `active-chat/` directory in your local browser
